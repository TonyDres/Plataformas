Otras plataformas

GitHub

Es un servicio que permite almacenar repositorios Git en la nube. En otras palabras, es como un Google Drive para proyectos de software donde puedes encontrar código.

IntelliJ IDEA

Es un entorno de desarrollo integrado. Un IDE que integra todas las herramientas que necesitas para desarrollar una plataforma, de modo que éste cuenta con un editor de código, un compilador, un depurador, etc.

Docker

Es una plataforma abierta para desarrolladores y administradores de sistemas para construir, enviar y utilizar aplicaciones distribuidas.

Jira

Es la herramienta de desarrollo de software número uno entre los equipos ágiles. Ayuda a planear, supervisar y gestionar proyectos de desarrollo de software.
